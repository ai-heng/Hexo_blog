常用软件快捷键

发表于 2017-07-22 22:00:00 | 分类于 快捷键 | | 阅读次数

前两天呢，自己用Visio作图工具画了一些流程图，之前自己也用它做过流程图，不过觉得操作上有些麻烦，所以不太喜欢，而这次从网上找了一些常用快捷键，发现效率真的高了很多，就喜欢上了这款软件。由此，我也感觉到去记一些常用快捷键还是挺有必要的，能够提升自己的用户体验，所以对于我经常使用的软件，我就大致整理一下平时最常用到的操作快捷键。我觉得这也就好比Markdown编译器，之前不熟悉它的时候，感觉着好像挺麻烦的，但是熟悉之后，发现格式完全可以在自己书写的过程中整理归纳好，自己可以全身心投入到写作当中，这种方式真的很棒，也就越来越喜欢使用它，对于快捷键也是如此，你不必再因为某个需求而在脱离键盘，还要用鼠标去操作，我觉得这样有助于自己提升精力。

ZendStudio常用快捷键

功能 快捷键
行注释 Crtl+/
块注释 Crtl+Shift+/
增加代码缩进 Tab
减少代码缩进 Shift+Tab
页面切换 Crtl+Tab
切换到前一个编辑的页面 Alt+←
切换到下一个编辑的页面 Alt+→
复制当前行到下一行 Crtl+Alt+↓
删除当前行 Crtl+D
编辑窗口最大化 Crtl+M
当前行所在位置到行尾 Shift+End
当前行所在位置到行首 Shift+Home
光标移动到行首 Home
光标移动到行尾 End
当前行与下一行互换位置 Alt+↓
当前行与上一行互换位置 Alt+↑
代码助手 Alt+/
跳转到函数所在位置 F3或（Crtl+函数）
在当前行的下一行插入空行 Shift+Enter
在当前行的上一行插入空行 Crtl+Shift+Enter

Chrome浏览器常用快捷键

功能 快捷键
关闭当前标签页 Crtl+w

CMD编译器常用快捷键

我之前已经觉得用Markdown编译器写起来已经很方便了，而昨天又注意到CMD编译器中还可以对Markdown语法进一步简化，因为它提供了常见语法的快捷键。

功能 快捷键 对于md语法
加粗 Crtl+B *粗体文本*
斜体 Crtl+I *斜体文本*
超链接 Crtl+L []()
段落引用 Crtl+Q >
行内代码 Crtl+K `此处输入代码`
插入图片 Crtl+G ![]()
有序列表 Crtl+O 1.,2.,3.,…
无序列表 Crtl+U -,*,+…

Editplus常用快捷键

功能 快捷键
切换到下一个文档窗口 Crtl+Tab
切换到上一个文档窗口 Crtl+Shift+Tab
切换到上次访问的文档窗口 F12
载入当前文档到浏览器 Crtl+B
选择当前行 Crtl+R
复制当前行到下一行 Crtl+J
删除当前行 Alt+Shift+Delete
新建普通文本文档 Crtl+N
打开一个现有的文档 Crtl+O
新建浏览器窗口 Crtl+Shift+B
是否显示所有文档窗口列表 F11

我平时也会经常使用OneNote，因为前几天总结了CMD编译器中的部分快捷键，今天在oneNote里去试了一下，发现有一些是通用的，然后我从网上找了找OneNote相关的快捷键，发现好多字体样式都可以用快捷键来实现，而且在Word中也是通用的，自己感觉就像发现了个宝似得，这样一来对于文字类的修饰会显得更加方便，我就想知道以前的自己都在想些什么，都不去开发快捷键，哈哈O(∩_∩)O…
